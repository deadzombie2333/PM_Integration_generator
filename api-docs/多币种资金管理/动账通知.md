# 动账通知

## 接口信息

**请求方式:** POST

**请求地址:** `https://pay-gate-uat.payermax.com/balanceNotifyUrl`

**接口说明:** 当商户因为充值、转账、换汇、提现操作产生账户余额变动的时候,我们将会通过该API进行通知。

### Body (application/json)

| 参数名 | 类型 | 必填 | 说明 | 限制 | 示例值 |
|--------|------|------|------|------|--------|
| code | string | 是 | 返回码,'APPLY_SUCCESS'代表有效通知 | <= 32 characters | APPLY_SUCCESS |
| msg | string | 是 | 有效通知描述:Success | <= 256 characters | Success |
| version | string | 是 | 接口版本,当前值为:1.0 | - | - |
| keyVersion | string | 是 | 密钥版本,当前值为:1 | <= 8 characters | - |
| appId | string | 是 | 商户应用Id,PayerMax分配给商户应用的唯一标识 | <= 64 characters | - |
| merchantNo | string | 否 | 商户号,商户与PayerMax业务签约时生成的唯一标识 | <= 32 characters | - |
| notifyTime | string | 是 | 通知时间,符合rfc3339规范,格式:yyyy-MM-dd'T'HH:mm:ss.SSSXXX | <= 32 characters | 2022-01-22T10:00:00.500+08:00 |
| notifyType | string | 是 | 通知类型,充值-DEPOSIT,汇兑-EXCHANGE,提现-WITHDRAW,转账-TRANSFER | <= 16 characters | DEPOSIT/EXCHANGE/WITHDRAW/TRANSFER |
| data | object | 是 | 数据体 | - | - |

#### data 对象

| 参数名 | 类型 | 必填 | 说明 | 限制 | 示例值 |
|--------|------|------|------|------|--------|
| amount | object | 是 | 动账金额信息 | - | - |
| counterparty | object | 是 | 交易对手方 | - | - |
| direction | string | 是 | 动账方向:INCREASE/DECREASE | - | - |
| relatedTradeNo | string | 是 | 关联PayerMax交易流水号 | <= 64 characters | TRANSNO001 |
| exchange | object | 否 | 汇率信息 | - | - |
| fee | object | 否 | 手续费 | - | - |

## Response

### 200 成功响应

#### Body (application/json)

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| msg | string | 是 | 响应消息 |
| code | string | 是 | 响应码 |
