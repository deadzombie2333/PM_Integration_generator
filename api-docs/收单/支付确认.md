## Request Body

**Content-Type:** `application/json`

| Parameter | Type | Required | Description | Constraints |
|-----------|------|----------|-------------|-------------|
| version | string | required | 接口版本 当前值为：1.4 | <= 8 characters |
| keyVersion | string | required | 密钥版本 当前值为：1 | <= 8 characters |
| requestTime | string | required | 请求时间，符合rfc3339规范，格式：yyyy-MM-dd'T'HH:mm:ss.SSSXXX 时间需要在当前时间两分钟内 | <= 32 characters |
| appId | string | required | 商户应用Id，PayerMax分配给商户应用的唯一标识 | <= 64 characters |
| merchantNo | string | optional | 商户号，商户与PayerMax业务签约时生成的唯一标识 | <= 32 characters |
| data | object | required | 请求数据体 | - |

### data Object

| Parameter | Type | Required | Description | Constraints |
|-----------|------|----------|-------------|-------------|
| outTradeNo | string | required | 原商户订单号 | <= 64 characters |
| tradeToken | string | optional | 正向交易单的交易Token | <= 64 characters |
| paymentDetail | object | optional | 退款回调地址，可后台配置【配置地址】 | - |

**Example:**
```json
{
  "version": "1.4",
  "keyVersion": "1",
  "requestTime": "2025-02-07T13:15:90.386Z",
  "appId": "your_app_id",
  "merchantNo": "your_merchant_no",
  "data": {
    "outTradeNo": "merchant_order_123",
    "tradeToken": "T2025020713159038685354",
    "paymentDetail": {}
  }
}
```

## Response

### 200 OK

**Content-Type:** `application/json`

| Parameter | Type | Required | Description | Constraints |
|-----------|------|----------|-------------|-------------|
| code | string | required | 返回码，'APPLY_SUCCESS'代表成功。只代表接口请求成功，不代表订单状态。 | - |
| msg | string | required | 返回描述，'Success.'。只代表接口请求成功，不代表订单状态。 | - |
| data | object | required | 响应数据体 | - |

### data Object

| Parameter | Type | Required | Description | Constraints |
|-----------|------|----------|-------------|-------------|
| outRefundNo | string | required | 商户退款申请号 | <= 64 characters |
| tradeOrderNo | string | required | 交易订单号 | <= 64 characters |
| refundTradeNo | string | required | PayerMax退款流水号 | <= 64 characters |
| status | string | required | 退款状态：REFUND_PENDING，REFUND_FAILED | <= 32 characters |

**Example:**
```json
{
  "code": "APPLY_SUCCESS",
  "msg": "Success.",
  "data": {
    "outRefundNo": "refund_123",
    "tradeOrderNo": "trade_456",
    "refundTradeNo": "pm_refund_789",
    "status": "REFUND_PENDING"
  }
}
```
