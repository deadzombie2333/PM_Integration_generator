## Request Body

**Content-Type:** `application/json`

| Parameter | Type | Required | Description | Constraints |
|-----------|------|----------|-------------|-------------|
| version | string | required | 接口版本 当前值为：1.4 | - |
| keyVersion | string | required | 密钥版本 当前值为：1 | - |
| requestTime | string | required | 请求时间，符合rfc3339规范，格式：yyyy-MM-dd'T'HH:mm:ss.SSSXXX 时间需要在当前时间两分钟内 | - |
| appId | string | required | 商户应用Id，PayerMax分配给商户应用的唯一标识 | - |
| merchantNo | string | optional | 商户号，商户与PayerMax业务签约时生成的唯一标识 | - |
| spMerchantNo | string or null | optional | ISV服务商的商户号 | Example: 010229810189302 |
| data | object | required | - | - |

### data Object

| Parameter | Type | Required | Description | Constraints |
|-----------|------|----------|-------------|-------------|
| batchId | string or null | required | 案件批次号，批量调单回复时必填,单笔调单回复时必须为空 | <= 64 characters, Example: P20210702104937 |
| caseId | string or null | required | 案件编号，单笔调单回复时必填，批量调单回复时必须为空 | <= 64 characters, Example: O20211118090801 |
| outUserId | string | required | 用户号 | <= 64 characters, Example: ZNW3e |
| responseContent | string | required | 回复内容信息 | <= 256 characters |
| representmentFile | array[string] | optional | 文件名称以案件批次号或案件编号命名，商户回复材料下载链接地址需填入此字段 | >= 1 items, <= 1024 items |

**Example Request (Single Case):**
```json
{
  "version": "1.4",
  "keyVersion": "1",
  "requestTime": "2025-01-07T13:15:90.386Z",
  "appId": "your_app_id",
  "merchantNo": "your_merchant_no",
  "spMerchantNo": null,
  "data": {
    "batchId": null,
    "caseId": "O20211118090801",
    "outUserId": "ZNW3e",
    "responseContent": "Merchant response with evidence",
    "representmentFile": [
      "http://xxx/O20211118090801.ZIP"
    ]
  }
}
```

**Example Request (Batch Cases):**
```json
{
  "version": "1.4",
  "keyVersion": "1",
  "requestTime": "2025-01-07T13:15:90.386Z",
  "appId": "your_app_id",
  "merchantNo": "your_merchant_no",
  "spMerchantNo": null,
  "data": {
    "batchId": "P20210702104937",
    "caseId": null,
    "outUserId": "ZNW3e",
    "responseContent": "Batch response with evidence",
    "representmentFile": [
      "http://xxx/O20211118090801.ZIP",
      "http://xxx/O20211118090802.ZIP"
    ]
  }
}
```

## Response

### 200 OK

**Content-Type:** `application/json`

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| code | string | required | 返回码，'APPLY_SUCCESS'代表成功 |
| msg | string | required | 返回描述，'Success.' |
| data | object | required | - |

### data Object

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| successCaseIds | array[string] or null | optional | 处理成功案件编号 |
| failCaseIds | array[string] or null | optional | 处理失败案件编号 |

**Example Response:**
```json
{
  "code": "APPLY_SUCCESS",
  "msg": "Success.",
  "data": {
    "successCaseIds": ["O20211118090801", "O20211118090802"],
    "failCaseIds": null
  }
}
```
