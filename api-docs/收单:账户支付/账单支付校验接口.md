# 账单支付校验接口 (Bill Payment Validation)

## Endpoint
```
POST https://pay-gate-uat.payermax.com/billPaymentValidationUrl
```

## Description
此接口仅适用于菲律宾当地的Bill Payment支付方式。PayerMax调用该接口请求商户获取本次支付操作的可行性（是否能够支付）。

## Request Body

**Content-Type:** `application/json`

| Parameter | Type | Required | Description | Constraints |
|-----------|------|----------|-------------|-------------|
| referenceNo | string | required | 商户的还款合同号，其中前五位由PayerMax生成。 | <= 64 characters |
| amount | number | required | 订单金额。 | - |
| currency | string | required | 订单币种。 | <= 3 characters |

**Example Request:**
```json
{
  "referenceNo": "12345ABC123",
  "amount": 1000.00,
  "currency": "PHP"
}
```

## Response

### 200 OK

**Content-Type:** `application/json`

| Parameter | Type | Required | Description | Constraints |
|-----------|------|----------|-------------|-------------|
| code | string | required | 返回码，'SUCCESS'代表成功 | Example: SUCCESS |
| msg | string | required | 返回描述，'Success' | Example: Success |
| data | object | optional | 响应data。 | - |

### data Object

| Parameter | Type | Required | Description | Constraints |
|-----------|------|----------|-------------|-------------|
| referenceNo | string | required | 商户的还款合同号，其中前五位由PayerMax生成。 | <= 64 characters |
| minAmount | number | required | 最小支付金额。 | - |
| maxAmount | number | required | 最大支付金额。 | - |
| currency | string | required | 订单币种 | <= 3 characters |
| confirmation | string | required | 校验结果，"Y"或"N"。 | <= 16 characters |
| failureReason | string | optional | 校验失败的原因。注意：confirmation为N时，该值必填。 | <= 255 characters |

**Example Response (Success):**
```json
{
  "code": "SUCCESS",
  "msg": "Success",
  "data": {
    "referenceNo": "12345ABC123",
    "minAmount": 100.00,
    "maxAmount": 50000.00,
    "currency": "PHP",
    "confirmation": "Y"
  }
}
```

**Example Response (Failure):**
```json
{
  "code": "SUCCESS",
  "msg": "Success",
  "data": {
    "referenceNo": "12345ABC123",
    "minAmount": 100.00,
    "maxAmount": 50000.00,
    "currency": "PHP",
    "confirmation": "N",
    "failureReason": "Invalid reference number or account not found"
  }
}
```
