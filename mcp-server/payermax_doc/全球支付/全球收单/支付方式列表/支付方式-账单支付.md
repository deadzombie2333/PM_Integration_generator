# 支付方式-账单支付

## 概述

该文档介绍PayerMax账单支付的支付方式能力概述。

PayerMax已支持菲律宾GCASH电子钱包、PALAWANPAY电子钱包、以及OTC线下便利店的账单支付。

## 支持的支付渠道

各付款渠道的能力可参考下方表格：

| 支付方式类型 | 支付渠道名称 | 国家/地区 | 币种 | 限额 | 相关能力 |
|------------|------------|---------|------|------|---------|
| BILL_PAYMENT | GCASH | 菲律宾 | PHP | 50-50000 | 退款、部分退款 |
| BILL_PAYMENT | PALAWANPAY | 菲律宾 | PHP | 50-50000 | 退款、部分退款 |
| BILL_PAYMENT | OTC | 菲律宾 | PHP | 50-50000 | 退款、部分退款 |

## 支付渠道详情

### GCASH
- **类型**: 电子钱包
- **国家**: 菲律宾
- **币种**: PHP
- **限额**: 50-50,000 PHP
- **支持功能**: 退款、部分退款

### PALAWANPAY
- **类型**: 电子钱包
- **国家**: 菲律宾
- **币种**: PHP
- **限额**: 50-50,000 PHP
- **支持功能**: 退款、部分退款

### OTC
- **类型**: 线下便利店
- **国家**: 菲律宾
- **币种**: PHP
- **限额**: 50-50,000 PHP
- **支持功能**: 退款、部分退款
