# 取消订阅计划

## 接口信息

**请求方式:** POST

**请求地址:** `https://pay-gate-uat.payermax.com/aggregate-pay/api/gateway/subscriptionCancel`

**接口说明:** 商户创建订阅计划后,可以主动取消订阅计划。但如果出现下面情况,则商户不能取消订阅计划:

(1) 该订阅计划处于激活中。

(2) 该订阅计划的某一期扣款处于支付中。
