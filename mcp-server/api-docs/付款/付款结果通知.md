# 付款结果通知

## 接口信息

**请求方式:** POST

**请求地址:** `https://pay-gate-uat.payermax.com/disbursementResultNotifyUrl`

**接口说明:** 该地址无需商户主动请求,以下request.body为payermax对订单状态的回调内容,您在收到回调后需要响应`{"msg": "Success","code": "SUCCESS"}`

### Body (application/json)

| 参数名 | 类型 | 必填 | 说明 | 限制 |
|--------|------|------|------|------|
| code | string | 是 | 回调通知码,APPLY_SUCCESS代表有效回调 | <= 32 characters |
| msg | string | 是 | 回调通知码描述,Success. | <= 256 characters |
| keyVersion | string | 是 | 密钥版本,当前版本1 | <= 8 characters |
| appId | string | 是 | 商户应用Id,PayerMax分配给商户应用的唯一标识 | <= 64 characters |
| merchantNo | string | 是 | 商户号,商户与PayerMax业务签约时生成的唯一标识 | <= 32 characters |
| notifyTime | string | 是 | 通知时间,符合rfc3339规范,格式:yyyy-MM-dd'T'HH:mm:ss.SSSXXX | <= 32 characters |
| notifyType | string | 是 | 通知类型 PAYOUT | <= 16 characters |
| data | object | 是 | 数据体 | - |

| 参数名 | 类型 | 必填 | 说明 | 限制 |
|--------|------|------|------|------|

| tradeNo | string | 是 | PayerMax交易流水号 | <= 64 characters |

| fundsReturnTime | string | 否 | 资金返还时间,符合rfc3339规范,格式:yyyy-MM-dd'T'HH:mm:ss Z | - |
| payFinishTime | string | 否 | 交易完成时间,符合rfc3339规范,格式:yyyy-MM-dd'T'HH:mm:ss Z | <= 32 characters |
| bounceBackTime | string | 否 | 退票时间,符合rfc3339规范,格式:yyyy-MM-dd'T'HH:mm:ss Z | <= 32 characters |
| redeemCode | string | 否 | 1.FAWRY取款码,2.运营商充值PIN码 | <= 32 characters |
| expiryTime | string | 否 | 订单过期时间,符合rfc3339规范,格式:yyyy-MM-dd'T'HH:mm:ss Z | <= 32 characters |

| source | object | 是 | 付款信息(商户) | - |

| reference | string | 否 | 附加数据,商户上送的值 | <= 512 characters |
| notifyPhone | string | 否 | 收款方通知邮件手机号 | <= 32 characters |
| notifyEmail | string | 否 | 收款方通知邮件地址 | <= 64 characters |

## Response

### 200 成功响应

#### Body (application/json)

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| msg | string | 是 | 响应 'Success'(仅代表成功收到通知,不代表订单状态) |
| code | string | 是 | 仅可响应 'SUCCESS'(仅代表成功收到通知,不代表订单状态) |
