# 风控欺诈通知

## 接口信息

**请求方式:** POST

**请求地址:** `https://pay-gate-uat.payermax.com/fraudNotifyUrl`

**接口说明:** 欺诈通知接口,用于通知商户产生了欺诈的交易,可用于商户自己对于欺诈交易或用户做风险管理

### Body (application/json)

| 参数名 | 类型 | 必填 | 说明 | 限制 | 示例值 |
|--------|------|------|------|------|--------|
| version | string | 是 | 接口版本,当前值为:1.0 | <= 8 characters | - |
| keyVersion | string | 是 | 密钥版本,当前值为:1 | <= 8 characters | - |
| notifyTime | string | 是 | 通知时间,符合rfc3339规范,格式:yyyy-MM-dd'T'HH:mm:ss.SSSXXX 时间需要在当前时间两分钟内 | <= 32 characters | - |
| appId | string | 是 | 商户应用Id,PayerMax分配给商户应用的唯一标识 | <= 64 characters | - |
| merchantNo | string | 否 | 商户号,商户与PayerMax业务签约时生成的唯一标识 | <= 32 characters | - |
| data | object | 是 | 请求数据体 | - | - |

#### data 对象

| 参数名 | 类型 | 必填 | 说明 | 限制 | 示例值 |
|--------|------|------|------|------|--------|
| merchantNo | string | 是 | 商户号唯一标识编号 | <= 32 characters | - |
| outTradeNo | string | 是 | 商户订单号 | <= 64 characters | 123456 |
| tradeAmount | string | 是 | 原交易金额 | <= 20 characters | 10.00 |
| tradeCurrency | string | 是 | 原交易币种 | <= 3 characters | IDR |
| tradeCreateTime | string | 是 | 原交易创建时间,符合rfc3339规范,格式:yyyy-MM-dd'T'HH:mm:ss.SSSXXX 时间需要在当前时间两分钟内 | <= 32 characters | 2024-01-22T10:00:00.500+08:00 |
| receiveDate | string | 否 | 欺诈接收时间,符合rfc3339规范,格式:yyyy-MM-dd'T'HH:mm:ss.SSSXXX 时间需要在当前时间两分钟内 | <= 32 characters | 2022-01-22T10:00:00.500+08:00 |
| outUserId | string | 是 | 商户外部用户唯一标识 | <= 64 characters | ZNW3e111 |
| requestTime | string | 是 | 请求时间,符合rfc3339规范,格式:yyyy-MM-dd'T'HH:mm:ss.SSSXXX 时间需要在当前时间两分钟内 | <= 32 characters | - |
| spMerchantNo | string | 否 | 平台商户号,非平台商户无需关注 | <= 32 characters | - |
| notifyType | string | 否 | 通知类型 ,当前默认值 FRAUD, 即指用户欺诈类通知 | <= 16 characters | FRAUD |
| code | string | 是 | 通知code,示例:APPLY_SUCCESS | <= 64 characters | APPLY_SUCCESS |
| msg | string | 是 | 通知msg,示例:Success. | <= 256 characters | Success. |

## Response

### 200 成功响应

#### Body (application/json)

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| code | string | 是 | 返回码,'SUCCESS'代表成功 |
| msg | string | 是 | 返回描述,'Success' |
