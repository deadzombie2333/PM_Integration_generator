# 换汇结果查询

## 接口信息

**请求方式:** POST

**请求地址:** `https://pay-gate-uat.payermax.com/aggregate-pay/api/gateway/exchangeQuery`

**接口说明:** 商户可以通过换汇结果查询接口,查询历史的换汇交易结果。

### Body (application/json)

| 参数名 | 类型 | 必填 | 说明 | 限制 |
|--------|------|------|------|------|
| version | string | 是 | 接口版本,当前值为:1.1 | - |
| keyVersion | string | 是 | 密钥版本,当前值为:1 | <= 8 characters |
| requestTime | string | 是 | 请求时间,符合rfc3339规范,格式:yyyy-MM-dd'T'HH:mm:ss.SSSXXX 时间需要在当前时间两分钟内 | <= 32 characters |
| appId | string | 是 | 商户应用Id,PayerMax分配给商户应用的唯一标识 | <= 64 characters |
| merchantNo | string | 否 | 商户号,商户与PayerMax业务签约时生成的唯一标识 | <= 32 characters |
| data | object | 是 | 请求数据体 | - |

#### data 对象

| 参数名 | 类型 | 必填 | 说明 | 限制 |
|--------|------|------|------|------|
| outTradeNo | string | 是 | 原商户订单号 | <= 64 characters |

## Response

### 200 成功响应

#### Body (application/json)

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| code | string | 是 | 返回码,'APPLY_SUCCESS'代表成功 |
| msg | string | 是 | 返回描述,'Success.' |
| data | object | 是 | 返回数据体 |

#### data 对象

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| transactionCcy | string | 是 | 交易币种 |
| side | string | 是 | 针对交易货币的买卖方向,买入BUY,卖出SELL |
| tradeNo | string | 是 | PayerMax交易流水号 |
| targetAmount | string | 是 | 到账金额 |
| rateTime | string | 是 | 汇率生成时间 |
| completeTime | string | 是 | 交易完成时间,符合rfc3339规范,格式:yyyy-MM-dd'T'HH:mm:ss.SSSXXX |
| errorCode | string | 是 | 订单失败错误码 |
| rateVersion | string | 是 | 汇率版本号 |
| rate | string | 是 | 汇率 |
| targetCcy | string | 是 | 目标币种 |
| errorMsg | string | 是 | 订单失败错误码描述 |
| ccyPair | string | 是 | 货币对 |
| outTradeNo | string | 是 | PayerMax交易流水号 |
| transactionAmount | string | 是 | 交易金额 |
| status | string | 是 | 交易状态,SUCCESS成功, FAILED失败,PENDING进行中; 终态才进行回调 |
